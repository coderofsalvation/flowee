Middleware

Flowee supports express/loopback/restify compatible middleware, using the familiar .use() function:

flowee.use( yourmiddleware );     
flowee.init(...)      
flowee.start(....){}

Basically anything goes into use(), as long as it looks like:

function (req,res,next){

}

NOTE: flowee.init(..) adds the http-router and fortunejs store-middleware. So the example will first process yourmiddleware before processing the other middleware.

Ratelimiting

Install ratelimit-middleware

npm install ratelimit-middleware

and then:

ratelimit = require('ratelimit-middleware');

flowee.use(ratelimit({
  burst: 10,
  rate: 0.5,
  ip: true
}));

flowee.init({model: model,store: true});

flowee.start( function(server, router) {
  var port;
  port = process.env.PORT || 1337;
  console.log("starting flowee at port %s", port);
  router.get('/foo', function(req, res, next) {
    return res.end("Hello foo");
  });
  return server.listen(port);
});

For more options see ratelimit-middleware
