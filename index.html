Flowee

Flowee is a very lightweight way to create api's:

  • built on the shoulders of fortunejs for automatic model<->relational database mapping
  • DDA: design-driven api by using a json model
  • SWAGGER v2 compatible json model
  • JSONAPI v1 compatible
  • middleware compatible: connect express and restify middleware modules from npm

Why

I took a look at loopback, meteor, cleverstack. They all have their pros/cons, which is fine. However, rolling your own polylithic api isn't that hard given the myriad of npm modules. Flowee likes to focus on:

  • lightweight
  • jsonmodel driven
  • no fileconventions
  • http framework agnostic
  • allow re-usage of middleware from http frameworks like express/restify etc
  • build your own cli tools
